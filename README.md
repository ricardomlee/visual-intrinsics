# Visual Intrinsics тАФ SIMD Register Explorer

An interactive, browser-based visualiser for x86 SSE/AVX-style 128/256/512-bit SIMD
registers.  The compute logic is written in **Rust**, compiled to **WebAssembly**
via [`wasm-pack`](https://rustwasm.github.io/wasm-pack/), and the UI is a
plain webpack app served from the `www/` directory.

> ЁЯУЦ Reference: [Intel Intrinsics Guide](https://www.intel.com/content/www/us/en/docs/intrinsics-guide/index.html)

## What it does

| Feature | Detail |
|---|---|
| **Three register widths** | Switch between XMM (`__m128i` 128-bit), YMM (`__m256i` 256-bit), and ZMM (`__m512i` 512-bit) |
| **Hex input** | Type any value (with or without `0x` prefix) and press **Set** |
| **Bit grid** | 4 rows ├Ч 32 columns of live bit cells, colour-coded by lane |
| **Multiple lane views** | Switch between `epi8`, `epu8`, `epi16`, `epu16`, `epi32`, `epu32`, `epi64`, or raw bits |
| **Editable lane values** | Click **тЬПя╕П Edit Lanes** to type directly into any A/B lane cell and apply the change immediately |
| **Bitwise ops** | AND, OR, XOR, ANDNOT (A op B), NOT A / NOT B |
| **Packed arithmetic** | Add, Sub, Saturating add/sub, Multiply (wrapping / saturating) |
| **Absolute value** | `abs_epi8/16/32` |
| **Compare & Select** | `cmpeq`, `cmpgt`, `max`/`min` signed and unsigned |
| **Shift** | Per-lane logical/arithmetic shift; full-register logical shift |
| **Rearrangement** | `unpacklo/hi`, `packs`/`packus`, `shuffle_epi8/32`, `alignr`, `blendv` |
| **Horizontal** | `hadd`/`hsub` epi16/32 |
| **Copy** | A тЖТ B, B тЖТ A, Result тЖТ A, Result тЖТ B |
| **Result display** | Hex value + lane table + bit grid for the computed result |

## Quick start

### Prerequisites

* [Rust toolchain](https://rustup.rs/) (stable, 1.70+)
* [`wasm-pack`](https://rustwasm.github.io/wasm-pack/installer/)  
  ```
  cargo install wasm-pack
  ```
* [Node.js](https://nodejs.org/) тЙе 18 + npm

### Build the WASM package

```
wasm-pack build
```

### Install web dependencies and run

```
cd www
npm install --legacy-peer-deps
npm start        # webpack dev server at http://localhost:8080
```

### Production build

```
cd www
npm run build    # output in www/dist/
```
