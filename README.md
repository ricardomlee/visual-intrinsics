# Visual Intrinsics — `__m128i` Explorer

An interactive, browser-based visualiser for x86 SSE/AVX-style 128-bit SIMD
registers.  The compute logic is written in **Rust**, compiled to **WebAssembly**
via [`wasm-pack`](https://rustwasm.github.io/wasm-pack/), and the UI is a
plain webpack app served from the `www/` directory.

## What it does

| Feature | Detail |
|---|---|
| **Two registers** | Side-by-side A and B panels, each representing one `__m128i` |
| **Hex input** | Type any 128-bit value (with or without `0x` prefix) and press **Set** |
| **Bit grid** | 4 rows × 32 columns of live bit cells, colour-coded by lane |
| **Multiple lane views** | Switch between `epi8` (16 × i8), `epi16` (8 × i16), `epi32` (4 × i32), `epi64` (2 × i64), or raw bits |
| **Bitwise ops** | AND, OR, XOR (A op B) |
| **Unary ops** | NOT A, NOT B |
| **Packed arithmetic** | `_mm_add_epi8`, `_mm_add_epi16`, `_mm_add_epi32` (wrapping) |
| **Shift** | Logical left / right shift of the full 128-bit value by any bit count |
| **Copy** | A → B, B → A, Result → A, Result → B |
| **Result display** | Hex value + lane table + bit grid for the computed result |

## Quick start

### Prerequisites

* [Rust toolchain](https://rustup.rs/) (stable, 1.70+)
* [`wasm-pack`](https://rustwasm.github.io/wasm-pack/installer/)  
  ```
  cargo install wasm-pack
  ```
* [Node.js](https://nodejs.org/) ≥ 18 + npm

### Build the WASM package

```
wasm-pack build
```

### Install web dependencies and run

```
cd www
npm install --legacy-peer-deps
npm start        # webpack dev server at http://localhost:8080
```

### Production build

```
cd www
npm run build    # output in www/dist/
```

## Project layout

```
src/
  lib.rs      — M128i struct + all Rust WASM API
  utils.rs    — panic hook helper
pkg/          — generated by wasm-pack (git-ignored)
www/
  index.html  — single-page UI
  index.js    — JS wiring to the WASM API
  bootstrap.js
  webpack.config.js
  package.json
tests/
  web.rs      — wasm-bindgen browser tests
```

## Running the browser tests

```
wasm-pack test --headless --chrome
```
